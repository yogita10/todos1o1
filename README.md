# How to see the App ? 
1. Clone the repository 
```sh
git clone https://github.com/yogita10/todos1o1.git
```
2. Go to the cloned folder
```sh
cd Food_Delivery_App
```
3. Run the following commands: 
```sh
 npm install 
```
```sh
 npm run android 
```
